# 🎯 才多 Duo 業務開發雲端系統 (v1.0)

## 📋 系統概述
本系統由 Codex AI 驅動，專為才多多同事開發，旨在將 104/1111 原始名單自動化清洗、去重並對接至 Notion 戰情室。

## 🚀 核心功能
1. **名單自動清洗**：自動解析原始 CSV，提取公司、聯絡人、電話。
2. **AI 去重與評級**：根據歷史資料自動標註重複項與潛力評級。
3. **Notion 自動對接**：清洗後的精華名單直接進入 [Notion 戰情室]。

## 🛠️ 同事使用 SOP
1. 將名單上傳至 [指定 Google 試算表]。
2. 每天 08:30 系統自動跑清洗。
3. 打開 Notion 手機版，直接點擊名單電話開始撥打。
